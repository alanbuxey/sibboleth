3. credentials – password management¶

3.1. SimpleCredentialManager Objects¶

class sibboleth.credentials.SimpleCredentialManager(username, password)¶

This class is responsible for authenticating the user in a non interactive way.

get_password()¶

return the password of the user

get_username()¶
prompt(controller)¶

Check that we havent tried to authenticate more then once, if we have then raise and exception

set_title(title)¶

show the title of the Basic Auth or Form that the user is presented with.

3.2. CredentialManager Objects¶

class sibboleth.credentials.CredentialManager(username=None, password=None)¶

This class is responsible for displaying information about the location being authed too, and receiving user name and password from the user.

get_password()¶

return the password of the user

get_username()¶

return the username of the user

prompt(controller)¶

the controller.run() function is called as the last method in this function to return control

Parameters:controller – the Shibboleth controller that control will be handed back to once the class is finshed taking input
set_title(title)¶

show the title of the Basic Auth or Form that the user is presented with.

3.3. Idp Objects¶

class sibboleth.credentials.Idp(idp=None)¶

This class responds to the WAYF form with the selected idp

get_idp()¶

return the selected idp

prompt(controller)¶

some how decide what idp to authenticate to the controller.run() function is called as the last method in this function to return control

Parameters:controller – the Shibboleth controller that control will be handed back to once the class is finshed taking input
set_idps(idps)¶

set the list of possible idps
